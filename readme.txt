== Description ==

Who Metrics is a free audience measurement service which reports demographics of website visitors and other metrics.

This plugin contains the necessary tags to load the Who Metrics service which is served externally from whometrics.net

Visit www.whometrics.com for all information about the service. All Who metrics settings are controlled via the website.

== Installation ==

Simply install the plugin and activate. There is zero configuration required.
